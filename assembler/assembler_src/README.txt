assembler.py:
    Take the given .asm file and converts it into 16-bit machine code 
    for hack ISA that is written into a new .hack file.

Usage:
$python assembler.py
    -shows usage and example
$python assembler.py <file-name>.asm
    -converts assembly code in the .asm file to machine code in a .hack file
